# Bhagwan Singh Vishwakarma Lecture Notes

Welcome to the **Bhagwan Singh Vishwakarma Lecture Notes** repository! This collection contains comprehensive, LaTeX-formatted digital notes for three major mathematical series. These notes have been systematically digitized, verified, and beautifully formatted to serve as high-quality study materials for exams.

## Lecture Series

The repository is divided into three distinct mathematical topics. Each topic has its own directory containing sequentially ordered lecture notes and a localized index for easy navigation.

### 1. [Matrix Theory](./Matrix/README.md)
This series covers the foundational elements of Matrix Theory, focusing on ranks, systems of linear equations, eigenvalues, eigenvectors, and the Cayley-Hamilton theorem.
- **Total Lectures**: 9

### 2. [Vector Calculus](./Vector_Calculus/README.md)
This series explores vector differentiation and integration. It includes profound treatments of Gradient, Divergence, Curl, and the three major integral theorems: Stokes' Theorem, Green's Theorem, and the Gauss Divergence Theorem.
- **Total Lectures**: 14

### 3. [Vector Space](./Vector_Space/README.md)
This extensive series dives deep into abstract algebra. It covers vector spaces, subspaces, linear independence, basis, dimension, linear transformations, quotient spaces, and matrix diagonalization.
- **Total Lectures**: 27

## Formatting and Standards

These notes strictly adhere to the repository's LaTeX mathematical formatting guidelines to ensure perfect rendering on GitHub and other Markdown viewers. 

*   **Block Math**: Used for all standalone equations, matrices, and multi-step derivations.
*   **Inline Math**: Used for variables, operators, and short mathematical phrases within textual sentences.
*   **Navigation**: Every lecture file includes simple `< Previous | Index | Next >` navigation links at both the top and bottom of the document to facilitate seamless studying.
