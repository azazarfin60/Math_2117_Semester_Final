# Linear Algebra

## Field Theory

A field is a non-empty set of elements with two binary operations. These operations are addition and multiplication. 

A binary operation maps two elements of a set to another element in the same set. Unary operations map a single element to another element.

### Field Axioms under Addition

For a set $F$ to be a field under addition, it must satisfy these five axioms:
1. **Closure**: For all $x, y \in F$, the sum $x+y$ is in $F$.
2. **Commutativity**: For all $x, y \in F$, $x+y = y+x$.
3. **Associativity**: For all $x, y, z \in F$, $(x+y)+z = x+(y+z)$.
4. **Identity**: There exists an element $0 \in F$ such that $0+x = x$ for all $x \in F$.
5. **Additive Inverse**: For every $x \in F$, there exists an element $-x \in F$ such that $x+(-x) = 0$.

If a set satisfies these properties, it is an abelian group under addition.

### Field Axioms under Multiplication

For the set $F$ to be a field under multiplication, the non-zero elements must satisfy these five axioms:
1. **Closure**: For all $x, y \in F$, the product $xy$ is in $F$.
2. **Commutativity**: For all $x, y \in F$, $xy = yx$.
3. **Associativity**: For all $x, y, z \in F$, $(xy)z = x(yz)$.
4. **Identity**: There exists a non-zero element $1 \in F$ such that $1 \cdot x = x$ for all $x \in F$.
5. **Multiplicative Inverse**: For every non-zero $x \in F$, there exists an element $x^{-1} \in F$ such that

$$
x \cdot x^{-1} = 1.
$$

Also, multiplication must distribute over addition:

$$
x(y+z) = xy + xz
$$

### Verification Examples

- **Set of Integers $\mathbb{Z}$**: The set of integers is a group under addition. But it is not a field. The multiplicative inverse of 2 is $1/2$, which is not an integer.
- **Set of Real Numbers $\mathbb{R}$**: The set of real numbers is a field. But we must exclude zero when checking multiplicative inverses. This is because $1/0$ is undefined.
- **Set of Rational Numbers $\mathbb{Q}$**: The set of rational numbers is a field.

## Vector Space and Subspace

A vector space $V$ is a set of vectors defined over a field $K$. It must be closed under vector addition and scalar multiplication.

### Vector Space Axioms

If $u, v, w \in V$ and $c, k \in K$, then a vector space must satisfy these rules:
1. **Vector Addition**: $u+v \in V$.
2. **Scalar Multiplication**: $ku \in V$.
3. **Commutativity of Addition**: $u+v = v+u$.
4. **Associativity of Addition**: $(u+v)+w = u+(v+w)$.
5. **Additive Identity**: There exists $0 \in V$ such that $u+0 = u$.
6. **Additive Inverse**: There exists $-u \in V$ such that $u+(-u) = 0$.
7. **Distributivity of Scalars**: $k(u+v) = ku + kv$.
8. **Distributivity of Vectors**: $(c+k)u = cu + ku$.
9. **Associativity of Scalar Multiplication**: $(ck)u = c(ku)$.
10. **Unit Multiplication**: $1u = u$.

### Examples and Non-Examples

- **Trivial Space**: The set $\{(0,0)\}$ containing only the zero vector is a vector space.
- **Euclidean Spaces**: $\mathbb{R}^2$, $\mathbb{R}^3$, and $\mathbb{R}^n$ are vector spaces.
- **Set of Straight Lines**: A straight line $y = mx$ passing through the origin is a vector space.
- **Non-Example 1**: The set $\{(1,2)\}$ is not a vector space. It is not closed under addition. For example, $(1,2) + (1,2) = (2,4)$, which is not in the set.
- **Non-Example 2**: The set $\{(1,2), (0,0), (-1,-2)\}$ is not a vector space. It is closed under addition. But it is not closed under scalar multiplication. For example, $3(1,2) = (3,6)$, which is not in the set.

### Subspaces

A subset $W$ of a vector space $V$ is a subspace if it satisfies three conditions:
1. The zero vector is in $W$. So $0 \in W$.
2. For all $u, v \in W$, the sum $u+v$ is in $W$.
3. For all $u \in W$ and $k \in K$, the scalar product $ku$ is in $W$.

Examples of subspaces:
- The set of all symmetric matrices is a subspace of the set of all square matrices.
- The set of even-power polynomials

$$
y = a\_2x^2 + a\_4x^4 + \dots
$$

is a subspace of the set of all polynomials.

## Linear Combinations, Span, and Basis

### Linear Combinations

A vector $v$ is a linear combination of vectors $v\_1, v\_2, \dots, v\_n$ if:

$$
v = c\_1v\_1 + c\_2v\_2 + \dots + c\_nv\_n
$$

Here, $c\_1, c\_2, \dots, c\_n$ are scalars.

### Spanning Set

The span of a set of vectors is the set of all their linear combinations.
If we can write any vector in $V$ using a set of vectors, then those vectors span $V$.
- In $\mathbb{R}^2$, standard vectors

$$
e\_1 = (1,0)
$$

and

$$
e\_2 = (0,1)
$$

span the space. Any vector can be written as

$$
(a,b) = ae\_1 + be\_2.
$$

- If vectors are linearly dependent, we need more vectors to span the space.

### Linear Independence

A set of vectors is linearly independent if the equation:

$$
c\_1v\_1 + c\_2v\_2 + \dots + c\_nv\_n = 0
$$

has only the trivial solution

$$
c\_1 = c\_2 = \dots = c\_n = 0
$$

. If there is a non-zero solution, the vectors are linearly dependent.

We can check linear independence using two methods:
1. **Method 1: Echelon matrix**: Form a matrix with the vectors as rows. Reduce it to echelon form. If the rank equals the number of vectors, they are independent.
2. **Method 2: Linear combination**: Write the linear combination equation and solve for the coefficients.

#### Matrix Independence Example

Let us check if these matrices are linearly independent:

$$
\sigma\_1 =
\begin{bmatrix}
2 & 2 \\
2 & 2
\end{bmatrix}, \quad \sigma\_2 = \begin{bmatrix} 2 & 0 \\\\ 0 & 2 \end{bmatrix}, \quad \sigma\_3 = \begin{bmatrix} 2 & 2 \\\\ 0 & 0 \end{bmatrix}
$$

We write the linear combination equation:

$$
c\_1 \sigma\_1 + c\_2 \sigma\_2 + c\_3 \sigma\_3 =
\begin{bmatrix}
0 & 0 \\
0 & 0
\end{bmatrix}
$$

This gives:

$$
c\_1
\begin{bmatrix}
2 & 2 \\
2 & 2
\end{bmatrix} + c\_2 \begin{bmatrix} 2 & 0 \\\\ 0 & 2 \end{bmatrix} + c\_3 \begin{bmatrix} 2 & 2 \\\\ 0 & 0 \end{bmatrix} = \begin{bmatrix} 0 & 0 \\\\ 0 & 0 \end{bmatrix}
$$

Combining the matrices:

$$
\begin{bmatrix}
2c\_1 + 2c\_2 + 2c\_3 & 2c\_1 + 2c\_3 \\
2c\_1 & 2c\_1 + 2c\_2
\end{bmatrix} = \begin{bmatrix} 0 & 0 \\\\ 0 & 0 \end{bmatrix}
$$

This yields a system of equations:
1.

$$
2c\_1 + 2c\_2 + 2c\_3 = 0 \Rightarrow c\_1 + c\_2 + c\_3 = 0
$$

2.

$$
2c\_1 + 2c\_3 = 0 \Rightarrow c\_1 + c\_3 = 0
$$

3.

$$
2c\_1 = 0 \Rightarrow c\_1 = 0
$$

4.

$$
2c\_1 + 2c\_2 = 0 \Rightarrow c\_1 + c\_2 = 0
$$

From equation (3), we have

$$
c\_1 = 0.
$$

Substituting

$$
c\_1 = 0
$$

in equation (4) gives

$$
c\_2 = 0.
$$

Substituting

$$
c\_1 = 0
$$

in equation (2) gives

$$
c\_3 = 0.
$$

Since all coefficients are zero, the matrices are linearly independent.

#### Vector Independence Rule

If the number of vectors is greater than the dimension of the space, they are always dependent. For example, four vectors in $\mathbb{R}^3$ are always dependent.

### Basis and Dimension

A set of vectors is a basis for a vector space $V$ if:
1. The vectors are linearly independent.
2. The vectors span $V$.

The standard basis for $\mathbb{R}^3$ is:

$$
\{(1,0,0), \ (0,1,0), \ (0,0,1)\}
$$

> [!NOTE]
> Linearly independent vector গুলোর সেটই হলো basis.

The dimension of a vector space is the number of vectors in its basis.

### Sum and Direct Sum

Let $U$ and $W$ be subspaces of $V$.
1. **Sum**: The sum is the set:

$$
U + W = \{ v = u + w \mid u \in U, \ w \in W \}
$$

The dimension formula is:

$$
\text{dim}(U + W) = \text{dim}(U) + \text{dim}(W) - \text{dim}(U \cap W)
$$

2. **Direct Sum**: If $U \cap W = \{0\}$, the sum is a direct sum. We write it as $U \oplus W$. The dimension of their intersection is zero.

## Linear Mappings

Let $U$ and $V$ be vector spaces. A mapping $T: U \rightarrow V$ is a linear mapping if:
1. **Superposition**: $T(u+v) = T(u) + T(v)$ for all $u, v \in U$.
2. **Homogeneity**: $T(ku) = k T(u)$ for all $u \in U$ and scalar $k$.

If $U = V$, the linear mapping is a linear operator.

> [!NOTE]
> সকল অপারেটর-ই ম্যাপিং কিন্তু সকল ম্যাপিং অপারেটর না।

### Linear Mapping Proofs

1. **Show that $f(x,y) = (2x, 2y)$ is a linear mapping.**
Let

$$
u = (a, b) \in \mathbb{R}^2
$$

and

$$
v = (a', b') \in \mathbb{R}^2.
$$

Check superposition:

$$
f(u+v) = f(a+a', b+b') = (2(a+a'), 2(b+b'))
$$

Also:

$$
f(u) + f(v) = (2a, 2b) + (2a', 2b') = (2(a+a'), 2(b+b'))
$$

So $f(u+v) = f(u) + f(v)$.
Check homogeneity:

$$
f(ku) = f(ka, kb) = (2ka, 2kb)
$$

Also:

$$
kf(u) = k(2a, 2b) = (2ka, 2kb)
$$

So $f(ku) = kf(u)$. The mapping is linear.

2. **Show that the projection mapping $f(x,y,z) = (x,y,0)$ is linear.**
Using a similar method, let $u = (a,b,c)$ and $v = (a',b',c')$.
Superposition:

$$
f(u+v) = f(a+a', b+b', c+c') = (a+a', b+b', 0)
$$

Also:

$$
f(u) + f(v) = (a,b,0) + (a',b',0) = (a+a', b+b', 0)
$$

So $f(u+v) = f(u) + f(v)$.
Homogeneity:

$$
f(ku) = f(ka, kb, kc) = (ka, kb, 0)
$$

Also:

$$
kf(u) = k(a,b,0) = (ka, kb, 0)
$$

So $f(ku) = kf(u)$. The projection mapping is linear.

3. **Show that $f(x,y) = (xy, x)$ is not a linear mapping.**
Let

$$
u = (a,b) \in \mathbb{R}^2.
$$

Check homogeneity:

$$
f(ku) = f(ka, kb) = (k^2ab, ka)
$$

Also:

$$
kf(u) = k(ab, a) = (kab, ka)
$$

Since $f(ku) \neq kf(u)$ for general $k$, homogeneity fails. So the mapping is not linear.

### Kernel and Image of a Linear Mapping

- **Image**: The image $\text{Im } T$ is the set of all output vectors.
- **Kernel**: The kernel $\text{Ker } T$ is the set of input vectors that map to the zero vector.

#### Kernel and Image Example

Let

$$
T: \mathbb{R}^3 \rightarrow \mathbb{R}^3
$$

be defined by:

$$
T(x,y,z) = (x+2y, \ y-z, \ x+2z)
$$

Let us find the basis and dimension of $\text{Im } T$ and $\text{Ker } T$.

1. **Finding $\text{Im } T$**:
The standard basis for $\mathbb{R}^3$ is $\{(1,0,0), (0,1,0), (0,0,1)\}$.
The images are:

$$
\begin{aligned}
T(1,0,0) &= (1,0,1) \\\\
T(0,1,0) &= (2,1,0) \\\\
T(0,0,1) &= (0,-1,2)
\end{aligned}
$$

We form a matrix with these image vectors and reduce it to echelon form:

$$
A =
\begin{bmatrix}
1 & 0 & 1 \\
2 & 1 & 0 \\
0 & -1 & 2
\end{bmatrix}
$$

Apply $R\_2 \rightarrow 2R\_1 - R\_2$:

$$
A \sim
\begin{bmatrix}
1 & 0 & 1 \\
0 & -1 & 2 \\
0 & -1 & 2
\end{bmatrix}
$$

Apply $R\_3 \rightarrow R\_2 - R\_3$:

$$
A \sim
\begin{bmatrix}
1 & 0 & 1 \\
0 & -1 & 2 \\
0 & 0 & 0
\end{bmatrix}
$$

The non-zero rows are $(1,0,1)$ and $(0,-1,2)$. They are independent.
So a basis for $\text{Im } T$ is:

$$
\{(1,0,1), \ (0,-1,2)\}
$$

The dimension is:

$$
\text{dim}(\text{Im } T) = 2
$$

2. **Finding $\text{Ker } T$**:
We solve $T(x,y,z) = (0,0,0)$:

$$
\begin{aligned}
x + 2y &= 0 \\\\
y - z &= 0 \\\\
x + 2z &= 0
\end{aligned}
$$

Using equations, subtract the third equation from the first:

$$
2y - 2z = 0 \Rightarrow y - z = 0
$$

This is the same as the second equation. So we have two independent equations:

$$
\begin{aligned}
x + 2y &= 0 \\\\
y - z &= 0
\end{aligned}
$$

We have 3 variables and 2 equations. So we have 1 free variable.
Let $z = 1$. Then $y = 1$ and $x = -2$.
So a basis for $\text{Ker } T$ is:

$$
\{(-2,1,1)\}
$$

The dimension is:

$$
\text{dim}(\text{Ker } T) = 1
$$

### Algebra of Linear Mappings Example

Let $T(x,y) = (x,0)$ and $T'(x,y) = (y,x)$ be linear mappings on $\mathbb{R}^2$.
Let us find:
1. $3T + 3T'(x,y) = 3(x,0) + 3(y,x) = (3x+3y, \ 3x)$
2. $TT'(x,y) = T(T'(x,y)) = T(y,x) = (y,0)$
3.

$$
T^2 T'(x,y) = T(T(T'(x,y))) = T(y,0) = (y,0)
$$

4.

$$
T T'^2(x,y) = T(T'(T'(x,y))) = T(T'(y,x)) = T(x,y) = (x,0)
$$

5.

$$
3T^2(x,y) = 3T(T(x,y)) = 3T(x,0) = (3x,0)
$$

## Inner Product Spaces

An inner product space is a vector space with an inner product operation $\langle u, v \rangle$.

### Inner Product Axioms

For a real vector space, the inner product must satisfy:
1. **Linearity**: $\langle au+v, \ w \rangle = a\langle u,w \rangle + \langle v,w \rangle$.
2. **Symmetry**: $\langle u, v \rangle = \langle v, u \rangle$.
3. **Positivity**: $\langle u, u \rangle \ge 0$.
4. **Positive Definiteness**: $\langle u, u \rangle = 0 \iff u = 0$.

For a complex space, the symmetry axiom becomes conjugate symmetry:

$$
\langle u, v \rangle = \overline{\langle v, u \rangle}
$$

### Norm and Distance

- **Norm**: The magnitude of a vector is:

$$
\|v\| = \sqrt{\langle v, v \rangle}
$$

- **Distance**: The distance between vectors $u$ and $v$ is:

$$
d(u,v) = \|u - v\|
$$

Let us find the distance between $u = (1,2,3)$ and $v = (4,5,6)$:

$$
d(u,v) = \sqrt{(1-4)^2 + (2-5)^2 + (3-6)^2} = \sqrt{(-3)^2 + (-3)^2 + (-3)^2} = \sqrt{27}
$$

- **Angle**: The angle between two vectors satisfies:

$$
\cos\theta = \frac{\langle u, v \rangle}{\|u\|\|v\|}
$$

### Verification Examples

1. **Orthogonal Basis of $\mathbb{R}^3$**:
Show that

$$
e\_1 = (1,0,0),
$$

$$
e\_2 = (0,1,0)
$$

, and

$$
e\_3 = (0,0,1)
$$

form an orthogonal basis.
First, check magnitudes:

$$
\|e\_1\| = 1, \quad \|e\_2\| = 1, \quad \|e\_3\| = 1
$$

Now check inner products:

$$
\langle e\_1, e\_2 \rangle = 0, \quad \langle e\_1, e\_3 \rangle = 0, \quad \langle e\_2, e\_3 \rangle = 0
$$

Since all magnitudes are 1 and inner products are 0, they form an orthonormal basis.

> [!NOTE]
> Orthogonal হলেই তারা একে অপরের linearly independent.

2. **Angle Between Matrices**:
Let us find the angle between:

$$
u =
\begin{bmatrix}
1 & 0 \\
0 & 1
\end{bmatrix}, \quad v = \begin{bmatrix} 1 & 1 \\\\ 1 & 1 \end{bmatrix}
$$

We define the inner product as the sum of products of corresponding elements:

$$
\langle u, v \rangle = 1(1) + 0(1) + 0(1) + 1(1) = 2
$$

Now find magnitudes:

$$
\|u\| = \sqrt{1^2 + 0^2 + 0^2 + 1^2} = \sqrt{2}, \quad \|v\| = \sqrt{1^2 + 1^2 + 1^2 + 1^2} = 2
$$

Using the formula:

$$
\cos\theta = \frac{\langle u, v \rangle}{\|u\|\|v\|} = \frac{2}{2\sqrt{2}} = \frac{1}{\sqrt{2}} \Rightarrow \theta = 45^\circ
$$

3. **Inner Product of Functions**:
We can define an inner product for functions over $[a,b]$ as:

$$
\langle f, g \rangle = \int\_a^b f(x)g(x) \, dx
$$

Let us check if $\sin x$ and $\cos x$ are orthogonal over $[-\pi/2, \pi/2]$:

$$
\langle \sin x, \cos x \rangle = \int\_{-\pi/2}^{\pi/2} \sin x \cos x \, dx = \left[ \frac{1}{2}\sin^2 x \right]\_{-\pi/2}^{\pi/2} = \frac{1}{2}(1 - 1) = 0
$$

The functions are orthogonal. This is the basis of Fourier series.
All elements of the Fourier basis:

$$
\{1, \ \sin x, \ \cos x, \ \sin 2x, \ \cos 2x, \ \dots\}
$$

are orthogonal.
