### Solved Problem 18

Let $P_1, P_2, P_3$ be points fixed relative to an origin $O$ and let $\mathbf{r}_1, \mathbf{r}_2, \mathbf{r}_3$ be position vectors from $O$ to each point. Show that if the vector equation $a_1\mathbf{r}_1 + a_2\mathbf{r}_2 + a_3\mathbf{r}_3 = \mathbf{0}$ holds with respect to origin $O$, then it will hold with respect to any other origin $O'$ if and only if $a_1 + a_2 + a_3 = 0$.

**Answer:**

> [!NOTE]
> **Figure Description (Page 9, Fig. 19):**
> A vector diagram showing two origins $O$ and $O'$. The position vectors from $O$ to points $P_1, P_2, P_3$ are $\mathbf{r}_1, \mathbf{r}_2, \mathbf{r}_3$. The position vectors from $O'$ to the same points are $\mathbf{r}'_1, \mathbf{r}'_2, \mathbf{r}'_3$. The vector from $O$ to $O'$ is $\mathbf{v}$.

Let $\mathbf{r}'_1, \mathbf{r}'_2, \mathbf{r}'_3$ be the position vectors of $P_1, P_2, P_3$ with respect to $O'$.
Let $\mathbf{v}$ be the position vector of $O'$ with respect to $O$.
From the figure:

$$
\mathbf{r}_1 = \mathbf{v} + \mathbf{r}'_1, \quad \mathbf{r}_2 = \mathbf{v} + \mathbf{r}'_2, \quad \mathbf{r}_3 = \mathbf{v} + \mathbf{r}'_3
$$

So the equation $a_1\mathbf{r}_1 + a_2\mathbf{r}_2 + a_3\mathbf{r}_3 = \mathbf{0}$ becomes:

$$
a_1(\mathbf{v} + \mathbf{r}'_1) + a_2(\mathbf{v} + \mathbf{r}'_2) + a_3(\mathbf{v} + \mathbf{r}'_3) = \mathbf{0}
$$

$$
(a_1 + a_2 + a_3)\mathbf{v} + a_1\mathbf{r}'_1 + a_2\mathbf{r}'_2 + a_3\mathbf{r}'_3 = \mathbf{0}
$$

This holds for any origin $O'$ (any vector $\mathbf{v}$) if and only if:

$$
a_1 + a_2 + a_3 = 0
$$

### Solved Problem 19

Find the equation of a straight line which passes through two given points $A$ and $B$ having position vectors $\mathbf{a}$ and $\mathbf{b}$ with respect to an origin $O$.

**Answer:**

> [!NOTE]
> **Figure Description (Page 9, Fig. 20):**
> A line passing through points $A$ (position vector $\mathbf{a}$) and $B$ (position vector $\mathbf{b}$) from origin $O$. A general point $P$ on the line has position vector $\mathbf{r}$.

Let $\mathbf{r}$ be the position vector of any point $P$ on the line.
From the figure:

$$
\mathbf{OA} + \mathbf{AP} = \mathbf{OP} \implies \mathbf{a} + \mathbf{AP} = \mathbf{r} \implies \mathbf{AP} = \mathbf{r} - \mathbf{a}
$$

$$
\mathbf{OA} + \mathbf{AB} = \mathbf{OB} \implies \mathbf{a} + \mathbf{AB} = \mathbf{b} \implies \mathbf{AB} = \mathbf{b} - \mathbf{a}
$$

Since $\mathbf{AP}$ and $\mathbf{AB}$ are collinear, $\mathbf{AP} = t\mathbf{AB}$, so:

$$
\mathbf{r} - \mathbf{a} = t(\mathbf{b} - \mathbf{a}) \quad \text{or} \quad \mathbf{r} = (1-t)\mathbf{a} + t\mathbf{b}
$$

### Solved Problem 21 **[Marked in Book]**

Prove that the magnitude $A$ of the vector $\mathbf{A} = A_1\mathbf{i} + A_2\mathbf{j} + A_3\mathbf{k}$ is:

$$
A = \sqrt{A_1^2 + A_2^2 + A_3^2}
$$

**Answer:**

> [!NOTE]
> **Figure Description (Page 10, Fig. 22):**
> A rectangular box in a Cartesian coordinate system with one corner at origin $O$ and the opposite corner at $P(A_1, A_2, A_3)$. The vector $\mathbf{A}$ is the diagonal $OP$.

By the Pythagorean theorem:

$$
(\overline{OP})^2 = (\overline{OQ})^2 + (\overline{QP})^2
$$

where $\overline{OQ}$ is the diagonal in the $xy$-plane:

$$
(\overline{OQ})^2 = (\overline{OR})^2 + (\overline{RQ})^2 = A_1^2 + A_2^2
$$

Since $\overline{QP} = A_3$:

$$
A^2 = A_1^2 + A_2^2 + A_3^2 \implies A = \sqrt{A_1^2 + A_2^2 + A_3^2}
$$

### Solved Problem 22 **[Marked in Book]**

Given $\mathbf{r}_1 = 3\mathbf{i} - 2\mathbf{j} + \mathbf{k}$, $\mathbf{r}_2 = 2\mathbf{i} - 4\mathbf{j} - 3\mathbf{k}$, $\mathbf{r}_3 = -\mathbf{i} + 2\mathbf{j} + 2\mathbf{k}$, find the magnitudes of:
(a) $\mathbf{r}_3$,
(b) $\mathbf{r}_1 + \mathbf{r}_2 + \mathbf{r}_3$,
(c) $2\mathbf{r}_1 - 3\mathbf{r}_2 - 5\mathbf{r}_3$.

**Answer:**

**(a) Magnitude of $\mathbf{r}_3$:**

$$
|\mathbf{r}_3| = |-\mathbf{i} + 2\mathbf{j} + 2\mathbf{k}| = \sqrt{(-1)^2 + 2^2 + 2^2} = \sqrt{9} = 3
$$

**(b) Magnitude of $\mathbf{r}_1 + \mathbf{r}_2 + \mathbf{r}_3$:**

$$
\mathbf{r}_1 + \mathbf{r}_2 + \mathbf{r}_3 = (3+2-1)\mathbf{i} + (-2-4+2)\mathbf{j} + (1-3+2)\mathbf{k} = 4\mathbf{i} - 4\mathbf{j} + 0\mathbf{k}
$$

$$
|\mathbf{r}_1 + \mathbf{r}_2 + \mathbf{r}_3| = \sqrt{4^2 + (-4)^2 + 0^2} = \sqrt{32} = 4\sqrt{2}
$$

**(c) Magnitude of $2\mathbf{r}_1 - 3\mathbf{r}_2 - 5\mathbf{r}_3$:**

$$
2\mathbf{r}_1 - 3\mathbf{r}_2 - 5\mathbf{r}_3 = 2(3\mathbf{i} - 2\mathbf{j} + \mathbf{k}) - 3(2\mathbf{i} - 4\mathbf{j} - 3\mathbf{k}) - 5(-\mathbf{i} + 2\mathbf{j} + 2\mathbf{k})
$$

$$
= (6 - 6 + 5)\mathbf{i} + (-4 + 12 - 10)\mathbf{j} + (2 + 9 - 10)\mathbf{k} = 5\mathbf{i} - 2\mathbf{j} + \mathbf{k}
$$

$$
|2\mathbf{r}_1 - 3\mathbf{r}_2 - 5\mathbf{r}_3| = \sqrt{5^2 + (-2)^2 + 1^2} = \sqrt{30}
$$

### Solved Problem 23 **[Marked in Book]**

If $\mathbf{r}_1 = 2\mathbf{i} - \mathbf{j} + \mathbf{k}$, $\mathbf{r}_2 = \mathbf{i} + 3\mathbf{j} - 2\mathbf{k}$, $\mathbf{r}_3 = -2\mathbf{i} + \mathbf{j} - 3\mathbf{k}$ and $\mathbf{r}_4 = 3\mathbf{i} + 2\mathbf{j} + 5\mathbf{k}$, find scalars $a, b, c$ such that $\mathbf{r}_4 = a\mathbf{r}_1 + b\mathbf{r}_2 + c\mathbf{r}_3$.

**Answer:**
We require:

$$
3\mathbf{i} + 2\mathbf{j} + 5\mathbf{k} = a(2\mathbf{i} - \mathbf{j} + \mathbf{k}) + b(\mathbf{i} + 3\mathbf{j} - 2\mathbf{k}) + c(-2\mathbf{i} + \mathbf{j} - 3\mathbf{k})
$$

$$
3\mathbf{i} + 2\mathbf{j} + 5\mathbf{k} = (2a + b - 2c)\mathbf{i} + (-a + 3b + c)\mathbf{j} + (a - 2b - 3c)\mathbf{k}
$$

Equating components:

$$
2a + b - 2c = 3
$$

$$
-a + 3b + c = 2
$$

$$
a - 2b - 3c = 5
$$

Solving this linear system gives $a = -2, b = 1, c = -3$. So $\mathbf{r}_4 = -2\mathbf{r}_1 + \mathbf{r}_2 - 3\mathbf{r}_3$.

### Solved Problem 24 **[Marked in Book]**

Find a unit vector parallel to the resultant of vectors $\mathbf{r}_1 = 2\mathbf{i} + 4\mathbf{j} - 5\mathbf{k}$, $\mathbf{r}_2 = \mathbf{i} + 2\mathbf{j} + 3\mathbf{k}$.

**Answer:**
Let the resultant be $\mathbf{R}$:

$$
\mathbf{R} = \mathbf{r}_1 + \mathbf{r}_2 = (2\mathbf{i} + 4\mathbf{j} - 5\mathbf{k}) + (\mathbf{i} + 2\mathbf{j} + 3\mathbf{k}) = 3\mathbf{i} + 6\mathbf{j} - 2\mathbf{k}
$$

The magnitude is:

$$
R = |\mathbf{R}| = \sqrt{3^2 + 6^2 + (-2)^2} = \sqrt{49} = 7
$$

The unit vector parallel to $\mathbf{R}$ is:

$$
\frac{\mathbf{R}}{R} = \frac{3\mathbf{i} + 6\mathbf{j} - 2\mathbf{k}}{7} = \frac{3}{7}\mathbf{i} + \frac{6}{7}\mathbf{j} - \frac{2}{7}\mathbf{k}
$$

### Solved Problem 25 **[Marked in Book]**

Determine the vector having initial point $P(x_1, y_1, z_1)$ and terminal point $Q(x_2, y_2, z_2)$ and find its magnitude.

**Answer:**

> [!NOTE]
> **Figure Description (Page 11, Fig. 23):**
> A coordinate system with points $P$ and $Q$ and their position vectors $\mathbf{r}_1$ and $\mathbf{r}_2$. The vector from $P$ to $Q$ is $\mathbf{PQ}$.

The position vector of $P$ is $\mathbf{r}_1 = x_1\mathbf{i} + y_1\mathbf{j} + z_1\mathbf{k}$.
The position vector of $Q$ is $\mathbf{r}_2 = x_2\mathbf{i} + y_2\mathbf{j} + z_2\mathbf{k}$.
By vector addition:

$$
\mathbf{r}_1 + \mathbf{PQ} = \mathbf{r}_2 \implies \mathbf{PQ} = \mathbf{r}_2 - \mathbf{r}_1 = (x_2 - x_1)\mathbf{i} + (y_2 - y_1)\mathbf{j} + (z_2 - z_1)\mathbf{k}
$$

The magnitude is:

$$
|\mathbf{PQ}| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}
$$

### Solved Problem 26

Forces $\mathbf{A}$, $\mathbf{B}$ and $\mathbf{C}$ acting on an object are given in terms of their components by the vector equations:

$$
\mathbf{A} = A_1\mathbf{i} + A_2\mathbf{j} + A_3\mathbf{k}
$$

$$
\mathbf{B} = B_1\mathbf{i} + B_2\mathbf{j} + B_3\mathbf{k}
$$

$$
\mathbf{C} = C_1\mathbf{i} + C_2\mathbf{j} + C_3\mathbf{k}
$$

Find the magnitude of the resultant of these forces.

**Answer:**
The resultant force is:

$$
\mathbf{R} = \mathbf{A} + \mathbf{B} + \mathbf{C} = (A_1 + B_1 + C_1)\mathbf{i} + (A_2 + B_2 + C_2)\mathbf{j} + (A_3 + B_3 + C_3)\mathbf{k}
$$

The magnitude of the resultant is:

$$
R = |\mathbf{R}| = \sqrt{(A_1 + B_1 + C_1)^2 + (A_2 + B_2 + C_2)^2 + (A_3 + B_3 + C_3)^2}
$$

### Solved Problem 27

Determine the angles $\alpha$, $\beta$ and $\gamma$ which the vector $\mathbf{r} = x\mathbf{i} + y\mathbf{j} + z\mathbf{k}$ makes with the positive directions of the coordinate axes and show that $\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1$.

**Answer:**

> [!NOTE]
> **Figure Description (Page 11, Fig. 24):**
> A Cartesian coordinate system showing a vector $\mathbf{r}$ extending to point $P(x,y,z)$. The diagonal projection angles $\alpha$, $\beta$, $\gamma$ from the axes are indicated.

Referring to the figure, triangle $OAP$ is a right triangle with right angle at $A$. So $\cos \alpha = \frac{x}{r}$.
Similarly, from right triangles $OBP$ and $OCP$:

$$
\cos \beta = \frac{y}{r}, \quad \cos \gamma = \frac{z}{r}
$$

where $r = |\mathbf{r}| = \sqrt{x^2 + y^2 + z^2}$.
Then:

$$
\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = \frac{x^2 + y^2 + z^2}{r^2} = 1
$$

The numbers $\cos \alpha$, $\cos \beta$, $\cos \gamma$ are called the direction cosines of the vector $\mathbf{OP}$.

### Solved Problem 28

Determine a set of equations for the straight line passing through the points $P(x_1, y_1, z_1)$ and $Q(x_2, y_2, z_2)$.

**Answer:**

> [!NOTE]
> **Figure Description (Page 12, Fig. 25):**
> A coordinate system with points $P$ and $Q$ on a straight line. The position vector of any point $R$ on the line is $\mathbf{r}$.

Let $\mathbf{r}_1$ and $\mathbf{r}_2$ be the position vectors of $P$ and $Q$ respectively. Let $\mathbf{r}$ be the position vector of any point $R$ on the line.
From the vector equation of the line (see Solved Problem 19):

$$
\mathbf{r} - \mathbf{r}_1 = t(\mathbf{r}_2 - \mathbf{r}_1)
$$

In rectangular components:

$$
(x - x_1)\mathbf{i} + (y - y_1)\mathbf{j} + (z - z_1)\mathbf{k} = t[(x_2 - x_1)\mathbf{i} + (y_2 - y_1)\mathbf{j} + (z_2 - z_1)\mathbf{k}]
$$

This gives the parametric equations of the line:

$$
x - x_1 = t(x_2 - x_1), \quad y - y_1 = t(y_2 - y_1), \quad z - z_1 = t(z_2 - z_1)
$$

Eliminating $t$, the equations become:

$$
\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1}
$$
